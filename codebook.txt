Information repository
=============================

This dataset contains all files needed for the week 4 submission of the Data Science Coursera course by John Hopkins
The relevant files are:
- run_analysis.R	: this contains all R scripts needed to produce the tidy output dataset
- output.txt		: this contains the tidy output dataset

==========================
